Plugin Name: Beekeeper
Goal: A configurable beekeeper NPC vendor and bee progression system for hardcore Rust servers.

Main Features:
1. Admin-placeable beekeeper NPC
2. Configurable shop
3. Honey selling system
4. Beehive progression
5. Custom bee-themed items
6. Configurable limits, prices, rewards, and permissions

V1 Features:
- Spawn beekeeper NPC with admin command
- Save NPC location
- Let players open beekeeper shop
- Sell configurable items
- Buy honey jars from players
- Admin-configurable prices
- Admin-configurable sell limits
- Admin can disable selling
- Admin can disable any shop item


Beehive Overhaul:
- Beehives have health
- Beehives can become sick
- Beehives can die
- Players must maintain hives
- Queen/Nucleus quality affects production
- Over-harvesting can damage the hive
- Gloves reduce damage and improve rare drop chance
