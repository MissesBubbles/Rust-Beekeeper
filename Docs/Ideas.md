# Beekeeper NPC

## Purpose
The Beekeeper is a configurable NPC that server admins can place anywhere in the world. The NPC acts as a merchant specializing in bee-related items and hive progression.

---

## Features

- Admin placeable
- Configurable name
- Configurable clothing
- Configurable dialogue
- Configurable shop inventory
- Buy items from players
- Sell items to players
- Honey buyback system
- Can be enabled/disabled
- Supports multiple NPCs (future)

---

## Admin Commands

/beekeeper spawn
/beekeeper remove
/beekeeper move
/beekeeper reload

---

## Player Interaction

- Open Shop
- Sell Honey
- Talk
